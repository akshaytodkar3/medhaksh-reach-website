# Medhaksh Reach — Website

Public marketing site for Medhaksh Reach. Static HTML/CSS/JS, no build step, no framework.

- `index.html` — homepage
- `privacy.html` — privacy policy

Separate from the main [MedhakshReach](../MedhakshReach) SaaS application repo — this is just the public-facing website (used for Meta Business Verification and general marketing).

## Deploy

Static site, zero-config on Vercel: connect this folder/repo to a new Vercel project and deploy — no build command or output directory needed.
